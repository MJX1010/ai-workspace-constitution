5. 不建议或禁止 Git 管理的内容

  • D:\Projects\.constitution-state.json：installer 状态文件。
  • *.backup.*：安装器或手工备份产物。
  • live 的 .codex\config.toml、.claude\settings.local.json：提交模板，不提交渲染结果。
  • .claude\skills\lark-doc、lark-shared junction：提交 lock，不提交链接结果。
  • 插件缓存、session、history、log、auth、credentials。
  • .memsearch\、.omc\、.hermes\ 等运行状态目录。
  • _codex_tmp\、_outputs\、_research\ 中的一次性产物。
